How to set up your custom new tab:

  1. Click the extension icon in your browser toolbar
  2. Enter your desired URL (web address or local file)
  3. Click Save

Examples:

  • Web URL: https://example.com
  • Local file (Windows): file:///C:/path/to/file.html
  • Local file (WSL): file://///wsl.localhost/Ubuntu/path/to/file.html
  • Local file (Mac): file:///Users/username/path/to/file.html

Important!

To access local files, you must enable "Allow access to file URLs" for this extension:

  1. Go to your browser's extensions page
  2. Find this extension and click "Details"
  3. Enable "Allow access to file URLs"
  4. Restart your browser
